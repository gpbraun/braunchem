**Assinale** a alternativa com o número de orbitais em subcamadas com número quântico $l$ igual a $0$, $1$, $2$ e $3$, respectivamente.

- [ ] $0$; $2$; $4$; $8$
- [ ] $1$; $2$; $3$; $4$
- [x] $1$; $3$; $5$; $7$
- [ ] $1$; $5$; $9$; $13$
- [ ] $2$; $6$; $10$; $14$

---

$$n_{\text{orbitais}}=\text{número de valores possíveis para }m_{l}$$
Para $l=0$ :
$$0\geq m_{l}\geq 0$$ 
1 valor possível 
$$n_{\text{orbitais}}=1$$
Para $l=1$ :
$$1\geq m_{l}\geq -1$$
3 valores possíveis
$$n_{\text{orbitais}}=3$$
Para $l=2$ :
$$2\geq m_{l}\geq -2$$
5 valores possíveis
$$n_{\text{orbitais}}=5$$
Para $l=3$ :
$$3\geq m_{l}\geq -3$$
7 valores possíveis
$$n_{\text{orbitais}}=7$$
De modo geral, para um $l$ fixo:
$$n_{\text{orbitais}}=2l+1$$
